# 靠山盟 AI 智能交付系统 - AGENTS 文档

## 概述

靠山盟 AI 智能交付系统是一个集成了多个专业 AI 代理的智能化平台，旨在帮助企业和个人在战略规划、内容创作、销售变现和产品交付等方面实现高效运营。

## 系统架构

系统采用模块化设计，将 AI 代理按照业务职能划分为四大部门：

- **战略与顶层设计部门**：负责商业定位、战略规划和品牌建设
- **内容与增长部门**：负责内容策划、创作和流量增长
- **销售与变现部门**：负责销售转化和商业变现
- **交付与产品部门**：负责产品验证和商业闭环

---

## 一、战略与顶层设计部门

### 1.1 定位诊断师

**功能描述**：
定位诊断师专注于帮助企业和个人进行精准的市场定位分析，识别核心竞争优势，制定差异化战略。

**核心能力**：
- 市场环境分析
- 竞争对手研究
- 目标用户画像构建
- 差异化定位建议
- 品牌价值主张提炼

**适用场景**：
- 新项目启动前的市场定位
- 现有业务的重新定位
- 品牌升级与转型
- 进入新市场的战略规划

**使用建议**：
1. 提供详细的业务背景信息
2. 明确当前面临的主要挑战
3. 说明目标市场和用户群体
4. 描述现有的竞争态势

---

### 1.2 商业操盘手

**功能描述**：
商业操盘手是您的战略顾问，帮助制定全局商业策略，规划业务增长路径，优化商业模式。

**核心能力**：
- 商业模式设计与优化
- 增长战略规划
- 资源配置建议
- 风险评估与应对
- 商业计划书撰写

**适用场景**：
- 创业初期的商业模式设计
- 业务扩张的战略规划
- 融资前的商业计划准备
- 业务转型的路径规划

**使用建议**：
1. 清晰描述业务现状和目标
2. 提供关键业务数据
3. 说明可用资源和约束条件
4. 明确时间节点和里程碑

---

### 1.3 IP人设定位

**功能描述**：
IP人设定位专家帮助打造独特的个人或品牌IP形象，建立差异化的人设标签，提升影响力。

**核心能力**：

- 人设标签设计
- 内容风格规划
- 传播策略制定
- IP价值变现路径

**适用场景**：
- 个人IP从0到1的打造
- 品牌IP形象升级
- 自媒体账号定位

**使用建议**：
1. 描述个人/品牌的核心优势
2. 明确目标受众群体
3. 说明期望的IP形象和调性

## 使用指南

### 如何选择合适的 AI 代理

1. **明确需求阶段**：
   - 战略规划阶段 → 战略与顶层设计部门
   - 内容创作阶段 → 内容与增长部门
   - 销售转化阶段 → 销售与变现部门
   - 产品优化阶段 → 交付与产品部门

2. **识别具体问题**：
   - 不清楚定位 → 定位诊断师
   - 需要商业策略 → 商业操盘手
   - 缺少内容选题 → 爆款选题策划
   - 文案转化率低 → 吸睛文案生成器
   - 私信成交困难 → 私信成交高手
   - 产品未验证 → MVP验证助手

3. **组合使用建议**：
   - 新项目启动：定位诊断师 → 商业操盘手 → MVP验证助手
   - 内容营销：爆款选题策划 → 吸睛文案生成器 → 流量分析
   - 销售转化：私信成交高手 → 话术生成师 → 商业闭环诊断

### 最佳实践

1. **提供充分的上下文信息**：越详细的背景信息，AI 代理能提供越精准的建议

2. **明确具体的目标**：清晰的目标有助于 AI 代理聚焦关键问题

3. **迭代优化**：根据 AI 代理的建议进行实践，再反馈结果进行优化

4. **跨部门协作**：复杂问题可以咨询多个 AI 代理，获得全方位的解决方案
